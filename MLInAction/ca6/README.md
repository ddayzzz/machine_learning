# SVM的代码
目前仅仅涉及二元SVC
# 实现的功能
1. SMO 优化算法
2. 核函数：高斯核函数（仅生成Gram矩阵和预测）、线性核函数（仅生成Gram矩阵）
3. 基于Tensorflow 的 SVM(目前使用的是高斯核函数)的线性不可分的数据集
